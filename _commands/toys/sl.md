sl
-------

Prints a train whenever you mistype ls as sl.

~~~ bash
$ sl
~~~

<!--more-->

It is like the example above, but better.

### How to install sl
~~~bash
$ sudo apt-get install sl
~~~

### Special flags
-a <br>
 the train flies

-l <br>
 a small train

-F <br>
 a flying train

-e <br>
 allows ctrl+c
